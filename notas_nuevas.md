- Me cuesta entender como pasar las cosas a github
  ```bash
  git add .
  git commit -m "Una descripción de los cambios"
  git push
  ```
  - `git add .`: agrega todos los cambios al área de preparación.
  - `git commit -m ""`: guarda los cambios en el historial con un mensaje descriptivo.
  - `git push`: envía los cambios al repositorio remoto.
